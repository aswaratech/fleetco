Token → spec mapping (this mockup invents nothing)

  • Sidebar: 240px expanded / 64px collapsed (DESIGN.md §Navigation). Item = 20px icon (size.icon.default) + 14px font-medium label. Active item background = color.accent.primary (#059669), text = color.accent.foreground (#ffffff).
  • Chrome surfaces: sidebar + top bar = color.surface.raised (#ffffff) with color.border.subtle (#e4e4e7) edges; content canvas = color.surface.canvas (#fafafa); table header band = color.surface.muted (#f4f4f5).
  • Top bar: sidebar collapse toggle, the ⌘K search affordance, and a user menu (avatar + email). The dropdown carries "Signed in as <email>" + "Sign out" — superseding the home dashboard's inline sign-out.
  • Breadcrumb + page header: breadcrumb (text-sm, color.text.muted, "›" separators) above an text-2xl font-semibold title, with the primary action right-aligned.
  • Primary button (focused, Frame 1): color.accent.primary background + a 3px color.border.focus ring at 50% (the single --focus-ring-width exception). Native selects use a color.border.strong (#d4d4d8) border.
  • Radius: controls + cards 4px (radius.default); popover / palette surfaces 8px (radius.lg); tables stay sharp.
  • Command palette: sources its destinations from the same shared nav source (lib/nav.ts) as the sidebar; navigation-only in Phase 1.
  • Voice: title-case group headings (not ALL-CAPS), factual labels, no exclamation marks (DESIGN.md §Voice).
